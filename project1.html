My Journey with HTML & CSS

This project marks my first deep dive into web design. With no background in HTML or CSS, I challenged myself to learn and create a responsive, visually appealing website. Every step was a lesson in design, code organization, and creative problem-solving.

Early Challenges: Getting Started

My initial attempts involved writing styling directly in the HTML. Quickly, I realized that mixing content with presentation was unsustainable. This phase was full of trial and error as I struggled with inline styling and debugging unexpected results.

Refactoring & Reorganizing

Realizing the need for clean code, I moved all styling to a dedicated CSS file. My CSS was initially chaotic—with redundancies and scattered declarations. Refactoring the stylesheet into clear, organized sections was challenging but essential for maintainability and further development.

Enhancements & Iterative Improvements

With a more organized codebase, I began integrating subtle visual enhancements—such as animations and parallax effects—to enrich the user experience. I also ensured the design was responsive and inherited dark mode settings from the main site. (Note: A scroll progress marker with interactive circles is planned for a future update.)

Final Product & Reflections

The final product is a responsive, visually appealing website that not only showcases my technical growth but also my ability to learn and adapt. This journey taught me the importance of clear code, thoughtful design, and continuous improvement. I’m proud of the progress made and excited for what comes next.
