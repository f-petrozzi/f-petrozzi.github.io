Building My Digital Presence

This project is my first experience with web development. With no prior knowledge of HTML or CSS, I challenged myself to create a responsive and visually appealing website. Each step taught me something new about design, code structure, and problem-solving. This website showcases my ability to learn and apply new skills through hands-on experience.

Early Challenges: Getting Started

Initially, I wrote all the styling directly in the HTML, thinking it would keep everything in one place. As the project grew, it became difficult to manage, and even small adjustments created unnecessary complications. Debugging was time-consuming, and maintaining consistency felt inefficient. Through trial and error, I better understood why separating content and presentation with CSS is essential for clean, scalable code.

Refactoring & Reorganizing

Understanding the importance of clean code, I transitioned all styling to a dedicated CSS file. At first, my stylesheet was unstructured, and filled with redundant and disorganized declarations. Organizing it into clear sections required careful refactoring, but it improved readability and made future updates more manageable. This process reinforced the value of structured code in maintaining efficiency and scalability.

Staying Focused on What Matters

With a structured codebase, I spent hours refining dark mode, adjusting animations, and experimenting with visual effects before adding any real content. It was easy to justify as polishing, but in reality, I was prioritizing design over substance. While the website is now visually appealing, the core sections still need to be built. Now, I am focusing on what should have come first by developing meaningful content, starting with this project.

Final Product & Reflections

The final product is a responsive and visually appealing website that reflects my progress in web development. Building it from the ground up challenged me to think critically about structure, design, and functionality. Along the way, I learned how to write cleaner code, make intentional design choices, and balance improvements with practicality. While there is always room for refinement, this project represents a strong foundation, and I look forward to applying what I have learned to future work.
