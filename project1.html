Building My Digital Presence

This project is my first experience with web development. With no background in HTML or CSS, I took on the challenge of building a responsive and visually engaging website.

Design...

Code Structure...

Problem Solving...

Every step was a learning experience, helping me improve my approach to design, structure, and problem-solving.

Getting Started

I started by writing all my styling directly in the HTML, thinking it would make everything easier to manage. As the project expanded, small adjustments became frustratingly complex.

Debugging...

Keeping everything consistent became nearly impossible, and it quickly became clear that mixing content with presentation made the code difficult to manage.

Learning Separation

Through trial and error, I learned the importance of separating content from styling to keep the code organized, scalable, and easier to maintain.

Refactoring & Reorganizing

I recognized the need for cleaner code and moved all styling to an external CSS file./p>

Refactoring...

My stylesheet was cluttered with redundant declarations, making navigation difficult. Reorganizing it into clear sections improved readability and made updating features much easier.

Efficiency & Scalability

This restructuring process reinforced the value of clean, scalable code.

Staying Focused on What Matters

With a structured codebase, I spent hours adjusting dark mode, animations, and visual effects to improve the user experience.

Visual Enhancements...

While the site looked more refined, the core content was still lacking and needed to be the priority.

Focusing on Substance

Now, I am focusing on adding projects that highlight different aspects of my skills and development.

Final Product & Reflections

The final product is a responsive and visually engaging website that came together through curiosity, experimentation, and problem-solving.

Elegant Design...

Building this site pushed me to think critically about structure, design, and functionality at every stage.

Clean Code & Functionality

This project provides a strong foundation to build on as I continue learning and expanding my skills.
